# Test app for Гексагон

Have to make frontend part of service for shortening links.

### How to launch

1. Setup `.env` by example:

```json
    REACT_APP_SERVER_ADRESS = http://XX.XXX.XX.XXX
```

2. `npm run start` for launching locally
